String

struct String

Extension for signing a String by converting it to utf8 Data and signing the bytes.

  • sign(with:)

    UTF8 encode the String to Data and sign it using the ECPrivateKey. The signing algorithm used is determined based on the private key’s elliptic curve.

    Throws

    An ECError if a valid signature is unable to be created.

    Return Value

    An ECSignature on failure.
