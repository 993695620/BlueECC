ECSignature

public struct ECSignature

The signature produced by applying Elliptic Curve Digital Signature Algorithm to some Plaintext data. It consist of two binary unsigned ints r and s.

  • r

    The r value of the signature. Will be 32 bytes of data for SHA256, 48 bytes for SHA384 or 66 bytes for SHA 512.
